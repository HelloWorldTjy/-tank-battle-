# 坦克大战 (Tank Battle)

一个基于 HTML5 Canvas 的经典坦克大战游戏，纯前端实现，无需任何依赖。

## 在线体验

直接用浏览器打开 `index.html` 即可开始游戏。

## 操作说明

| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` | 移动坦克 |
| `空格` | 发射炮弹 |
| `P` | 暂停/继续 |

## 游戏玩法

- 控制黄色坦克消灭所有敌方坦克（共 20 辆）
- 保护底部的基地（标有"基"的方块）
- 砖墙可被打穿，钢墙不可破坏
- 每消灭一辆敌军得 100 分

## 项目结构

```
tank-battle/
├── index.html   # 游戏主文件（包含 HTML + CSS + JS）
└── README.md    # 项目说明
```

## 技术栈

- HTML5 Canvas
- Vanilla JavaScript (ES6+)
- 纯 CSS 样式

## 本地运行

```bash
# 直接用浏览器打开
open index.html

# 或启动本地服务器
npx serve .
```
